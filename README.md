# Task Tracker

Task Tracker is a simple command-line application for managing tasks.  
It allows you to add, update, delete, complete, and list tasks easily.  
All tasks are stored in a JSON file for persistence.

Note: This project does not require any external dependencies. All modules used are part of Python's standard library.

---

## Features

- Add new tasks  
- Update existing tasks  
- Mark tasks as completed  
- Delete tasks  
- List all tasks
